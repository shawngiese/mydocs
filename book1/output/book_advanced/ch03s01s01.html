1.1. Embed images in data-uri

This is built-in to AsciiDoc and enables you to convert your PNG images into base64 encodings that are included in the HTML output file. When you use this feature, the image data is included in the asciidoc file and avoids having to distribute the images files. For more information read the AsciiDoc documentation for instructions using :data-uri:.
